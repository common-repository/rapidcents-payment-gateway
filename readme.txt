RapidCents Payment Gateway enables secure credit card payments with 3D Secure in WooCommerce.

== Description ==
RapidCents Payment Gateway Plugin enables WooCommerce store owners to accept secure credit card payments with 3D Secure authentication. It offers seamless integration, enhanced security, and a user-friendly interface for both admins and customers.

**Key Features:**
- Accept payments via credit card with 3D Secure.
- Reduce fraud and chargebacks with additional security.
- Simple integration and easy configuration.
- Test mode for transaction simulation.
- IP whitelisting for secure transaction processing.

== Installation ==
1. Download the plugin from the WordPress plugin repository or from the RapidCents website.
2. Log in to your WordPress admin panel.
3. Navigate to **Plugins > Add New**.
4. Click **Upload Plugin** and select the plugin ZIP file.
5. Click **Install Now** and then **Activate**.
6. Once activated, configure the plugin by navigating to **WooCommerce > Settings > Payments** and selecting **RapidCents**.
